# PromptLab Play Store Data Safety 초안

이 문서는 PromptLab을 Google Play Store에 제출하기 전에 Play Console의 Data Safety 섹션을 작성하기 위한 내부 기준 문서입니다.

운영 사이트: https://promptlab.io.kr  
개인정보 처리방침: https://promptlab.io.kr/privacy

## 1. 앱 개요

PromptLab은 사용자가 프롬프트를 작성, 저장, 수정, 공개, 보관하고, 저장 전 AI SafeCheck로 위험 요소를 검사할 수 있는 AI 프롬프트 작업실입니다.

현재 앱은 무료 MVP로 운영되며, 유료 결제, 인앱 구매, 구독 기능은 제공하지 않습니다.

## 2. 수집하는 데이터

PromptLab은 서비스 제공을 위해 다음 데이터를 처리할 수 있습니다.

- 이메일 주소
- 소셜 로그인 식별 정보
- 사용자가 직접 저장한 프롬프트 정보
- SafeCheck 검사 결과 메타데이터

프롬프트 정보에는 제목, 카테고리, 사용 목적, 본문, 예시 입력, 예시 출력, 안전 주의사항, 공개 범위, 상태가 포함될 수 있습니다.

SafeCheck 리포트에는 위험 점수, 최종 판정, 위험 카테고리, 안전 문장 안내, 정책 버전, 탐지기 버전, 검사 시각만 저장합니다.

## 3. 저장하지 않는 데이터

PromptLab은 SafeCheck 리포트에 검사 원문을 저장하지 않습니다.

저장하지 않는 데이터는 다음과 같습니다.

- 검사 원문 프롬프트
- 고객명
- 전화번호
- 이메일 원문
- 회사기밀 원문
- 내부자료 원문
- 계약조건 원문
- 상담기록, 진료기록 같은 민감한 원문

## 4. 데이터 이용 목적

수집한 데이터는 다음 목적으로 사용합니다.

- 사용자 로그인
- 사용자별 프롬프트 저장과 관리
- 프롬프트 공개 여부 관리
- SafeCheck 검사 결과 제공
- SafeCheck 리포트 조회
- 서비스 안정성 유지

## 5. 제3자 서비스

PromptLab은 다음 외부 서비스를 사용합니다.

| 서비스 | 사용 목적 |
|---|---|
| Supabase | 사용자 인증, 세션 관리, 데이터베이스 저장 |
| Google OAuth | Google 계정 로그인 |
| Kakao OAuth | Kakao 계정 로그인 |
| Vercel | 웹앱 배포와 호스팅 |
| GitHub | 소스 코드 버전 관리 |

PromptLab은 사용자 데이터를 광고 판매나 데이터 브로커 제공 목적으로 공유하지 않습니다.

## 6. 보안

PromptLab은 다음 보안 기준을 적용합니다.

- HTTPS 사용
- Supabase Auth 사용
- service_role key 클라이언트 노출 금지
- 로그인한 사용자만 본인 프롬프트 관리 가능
- 공개 화면에서 작성자 이메일과 사용자 ID 숨김
- SafeCheck 검사 원문 리포트 미저장
- block 판정 프롬프트 저장 차단
- review 판정 프롬프트 비공개 초안 저장만 허용

## 7. Play Console 입력 기준

### 앱이 사용자 데이터를 수집하나요

예.

로그인, 사용자별 프롬프트 저장, SafeCheck 리포트 제공을 위해 이메일 주소와 사용자 콘텐츠를 처리합니다.

### 앱이 데이터를 공유하나요

광고, 판매, 데이터 브로커 목적 공유는 하지 않습니다.

서비스 운영을 위해 Supabase, Google OAuth, Kakao OAuth, Vercel 같은 서비스 제공자가 데이터를 처리할 수 있습니다.

### 수집하는 데이터 유형

- 개인 정보: 이메일 주소
- 사용자 콘텐츠: 사용자가 저장한 프롬프트 정보
- 앱 활동: SafeCheck 검사 결과 메타데이터

### 수집 목적

- 앱 기능
- 계정 관리
- 사용자별 데이터 저장
- 보안 검사
- 서비스 운영

### 전송 중 데이터 암호화

예. HTTPS를 사용합니다.

### 데이터 삭제 요청

가능합니다.

사용자는 본인 프롬프트를 수정하거나 보관할 수 있으며, 계정 또는 저장 데이터 삭제가 필요한 경우 운영자 이메일로 요청할 수 있습니다.

### 결제 정보

수집하지 않습니다.

현재 PromptLab은 무료 MVP이며 유료 결제, 인앱 구매, 구독 기능을 제공하지 않습니다.

### 어린이 대상 여부

아니오.

PromptLab은 일반 사용자를 대상으로 한 생산성, 교육, 유틸리티 성격의 서비스이며 어린이를 주요 대상으로 하지 않습니다.
